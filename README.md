# Austria Election Data Analysis

## Project Goal
Analyze Austrian election results to validate the dataset, calculate seat allocations, and identify regional voting patterns.

## Methods
- Data validation and plausibility checks
- Calculation of seat allocation using the D'Hondt method
- Statistical analysis of party performance across federal states
- Data visualization

## Tools
- R
- Data visualization libraries in R

## Key Insights
- Identification of the strongest and weakest regions for each party
- Visualization of seat distribution and voting patterns
